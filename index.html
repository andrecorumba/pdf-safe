Passo a passo

Como funciona

Três passos simples para proteger seus documentos antes de compartilhá-los.

01

Abra o documento

Arraste ou selecione o arquivo PDF no seu computador. O arquivo é carregado diretamente no navegador — nenhum upload acontece.

02

Revise as detecções

O sistema identifica automaticamente os dados sensíveis e exibe uma lista para revisão. Marque ou desmarque os itens conforme necessário.

03

Baixe o PDF sanitizado

Com um clique, gere e baixe o documento com todos os dados sensíveis selecionados ocultados permanentemente.
